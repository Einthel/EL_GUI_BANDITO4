# Changelog — плагин Shortcut

Все изменения, связанные с плагином **Shortcut**, вынесенные из основного [changelog.md](changelog.md).

Формат: [Keep a Changelog](https://keepachangelog.com/en/1.0.0/), версии — [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

---

## [0.0.1.11] - 2026-02-13

### Fixed
- **Core (Cliento):** Исправлена логика автоматического поиска скомпилированного UI; теперь клиент корректно находит `ui_*_cliento.py` в подпапке `resources/ui_done/`, что критично для плагинов `music` и `shortcut`.

## [0.0.1.10] - 2026-02-10

### Added
- **Диалог выбора приложения (Application):**
  - Реализована форма выбора программы (`AppListDialog`, `ui_app_list_di.py`): сканирование ярлыков из меню «Пуск», отображение списка в `app_listW`, выбор через кнопку `select_toolB`.
  - Кнопка `app_choice_toolB` на вкладке Application открывает диалог; выбранный путь отображается в поле `app_path_lineE` (только для чтения).
  - При сохранении пресета с активной вкладкой Application формируется кнопка типа `program` с путём к приложению и отображаемым именем (`app_title_lineE`).
  - При загрузке настроек кнопки типа `program` поле `app_path_lineE` заполняется сохранённым путём.

---

## [0.0.1.9] - 2026-02-10 (Shortcut & Core)

### Changed
- **Рефакторинг версий плагина Shortcut:**
  - Устранён конфликт имён файлов версий: `plugins/shortcut/ver` → `ver_shortcut`.
  - Обновлены манифесты и логика апдейтеров для поддержки новых имён.

### Added
- **Система префабов (Shortcut):**
  - Реализована библиотека шаблонов кнопок (префабов), хранящаяся в `prefab_but_shortcut.json`.
  - **UI дерева:** `TreeWidget` отображает список доступных префабов, сгруппированных по категориям (Application, Shortcut, Action, Other).
  - **Сохранение:** При создании новой кнопки открывается диалог `SavePrefabDialog` для сохранения кнопки как именованного шаблона.
  - **Валидация:** Проверка уникальности имени префаба с диалогом подтверждения перезаписи (`QuestionsDialog`).
  - **Управление:** Контекстное меню в дереве для удаления и переименования префабов.
- **Push-обновление контента (Live Update) для Shortcut:**
  - Мгновенная отправка новых иконок на клиент (`SHORTCUT_ICON_UPDATE`) при сохранении.
  - Мгновенная отправка обновлённого конфига (`SHORTCUT_CONFIG_UPDATE`).
  - Клиент принимает и сохраняет изменения без перезагрузки приложения.

## [0.0.1.8] - 2026-02-10

### Added
- **UI & UX (shortcut_bandito.py):**
  - Поддержка выбора иконок в форматах: `.png`, `.jpg`, `.ico`, `.svg`.
  - Контекстное меню для кнопок с опцией «Delete» для очистки конфигурации кнопки.
- **Client Logic (shortcut_cliento.py):**
  - Динамическое создание страниц: плагин автоматически генерирует недостающие страницы и кнопки на основе конфигурации.
  - Локальная навигация по страницам («Page Prev», «Page Next») на клиенте без запросов к серверу.
- **Configuration:**
  - В `config_shortcut.json` добавлены системные действия «Page Prev» и «Page Next».
  - Редактор обновлён для сохранения системных действий через вкладку «Action».

---

## [0.0.1.7] - 2026-02-10

### Added
- **Редактор кнопок (shortcut_bandito.py):**
    - Реализовано управление страницами: добавление (`+`), удаление (`-`), навигация (`Next`/`Back`).
    - Добавлено сохранение конфигурации страниц в `button_shortcut.json` (кнопка `Save Change`).
    - Реализовано создание и сохранение новых пресетов кнопок (Имя + Хоткей) через кнопку `Save Preset`.
    - Поля ввода имени и хоткея теперь синхронизированы с выбранной кнопкой и обновляются в реальном времени.

## [0.0.1.6] - 2026-02-09

### Added
- **Логика плагина (shortcut_cliento.py):**
  - Обработка нажатий на кнопки.
  - Отправка команд `PLUGIN_BUTTON_PRESS` на сервер с ID кнопки (формат `page:button_id`).
  - Файл конфигурации `button_shortcut.json` для описания кнопок (внешний вид + действия).
- **Серверная часть (shortcut_bandito.py):**
  - Реализована обработка команды `PLUGIN_BUTTON_PRESS`.
  - Поддержка действий:
    - `program`: запуск внешних приложений (exe);
    - `shortcut`: эмуляция нажатия клавиш (библиотека `keyboard`).
- **Manifest:**
  - Добавлен `button_shortcut.json` в список синхронизируемых файлов.
- **Интерфейс редактора (shortcut_bandito.py):**
    - Реализован класс `ShortcutBanditoPlugin`, наследующий UI.
    - Внедрена логика активации/деактивации панели настроек в зависимости от выбора кнопки.
    - Реализована синхронизация кнопки предпросмотра (`Example_toolB`) с выбранным элементом.
    - Поддержка `style_shortcut.json` для визуализации состояния кнопок (Checked/Pressed).
- **Обновление плагина Shortcut:**
  - Обновлён манифест плагина (`plugins/shortcut/shortcut_manifest.json`) с корректными путями.
