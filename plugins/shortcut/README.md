# Plugin: Shortcut

Плагин «Stream Deck»: кнопки с действиями (запуск программ, горячие клавиши), страницы, префабы, иконки.

## Сервер (Bandito)

* **Редактор кнопок (shortcut_bandito.py):**
  * Управление страницами: добавление (`+`), удаление (`-`), навигация (Next/Back).
  * Сохранение конфигурации страниц в `button_shortcut.json` (кнопка Save Change).
  * Создание и сохранение пресетов (Имя + Хоткей) через Save Preset.
  * Поля имени и хоткея синхронизированы с выбранной кнопкой в реальном времени.
* **Система префабов:**
  * Библиотека шаблонов в `prefab_but_shortcut.json`.
  * TreeWidget: префабы по категориям (Application, Shortcut, Action, Other).
  * Сохранение кнопки как префаба (`SavePrefabDialog`), проверка уникальности имени (`QuestionsDialog`).
  * Контекстное меню: удаление и переименование префабов.
* **Вкладка Application:**
  * Диалог выбора программы (`AppListDialog`, `ui_app_list_di.py`): сканирование ярлыков из меню «Пуск», список в `app_listW`, выбор через `select_toolB`.
  * Кнопка `app_choice_toolB` открывает диалог; путь в `app_path_lineE` (read-only).
  * Сохранение пресета с вкладкой Application создаёт кнопку типа `program` (путь + отображаемое имя `app_title_lineE`).
* **Иконки:** форматы `.png`, `.jpg`, `.ico`, `.svg`; контекстное меню кнопок — опция Delete для очистки конфигурации.
* **Действия (вкладка Action):** системные «Page Prev» и «Page Next» сохраняются в `config_shortcut.json`.
* **Обработка на клиенте:** команда `PLUGIN_BUTTON_PRESS` (ID кнопки `page:button_id`), делегирование в плагин; действия:
  * `program` — запуск внешних приложений (exe).
  * `shortcut` — эмуляция нажатий клавиш (библиотека `keyboard`).
* **Стили:** `style_shortcut.json` для состояний кнопок (Checked/Pressed).

## Клиент (Cliento)

* **shortcut_cliento.py:**
  * Динамическое создание страниц и кнопок по конфигурации.
  * Локальная навигация по страницам (Page Prev / Page Next) без запросов к серверу.
  * Обработка нажатий: отправка `PLUGIN_BUTTON_PRESS` на сервер.

## Конфигурация и данные

* `button_shortcut.json` — описание кнопок (внешний вид + действия), синхронизируется с клиентом.
* `config_shortcut.json` — системные действия (Page Prev/Next) и настройки.
* `prefab_but_shortcut.json` — библиотека префабов.

## Live Update

* При сохранении на сервере:
  * Мгновенная отправка новых иконок на клиент (`SHORTCUT_ICON_UPDATE`).
  * Мгновенная отправка обновлённого конфига (`SHORTCUT_CONFIG_UPDATE`).
* Клиент применяет изменения без перезапуска приложения.
